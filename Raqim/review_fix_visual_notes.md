# ملاحظات مرئية بعد معاينة الصور المرفقة

الصورة الأولى المعاينة (`pasted_file_H4yWe6_image.png`) تعرض شاشة مراجعة رقيم المقسمة إلى عمودين: الملف الأصلي على اليمين والنص المستخرج على اليسار، مع قائمة اقتراحات صغيرة بجانب الكلمة. هذا يؤكد أن المطلوب هو إبقاء واجهة المراجعة جنبًا إلى جنب وعدم استخدام نافذة منبثقة كبيرة.

الصورة الثانية (`pasted_file_e39XSe_image.png`) تعرض جزءًا من README الخاص بـ CorpusFilter. التعليمات المرئية تقول إن التشغيل المحلي يتم عبر دخول مجلد `api`، تثبيت المتطلبات، ثم دخول `src` وتشغيل `python -m app`. كما تعرض خيار Docker ببناء صورة من داخل مجلد `api`.
