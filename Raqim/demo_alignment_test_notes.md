# ملاحظات اختبار تظليل الإحداثيات في الديمو

تم تشغيل نسخة الواجهة المعدلة بنجاح بعد البناء، وفتح صفحة المراجعة على الرابط المؤقت. عند الضغط على كلمة "الدرس" في النص المستخرج، ظهر إطار تحديد أوضح على معاينة الملف الأصلي وانتقلت المعاينة إلى أعلى موضع الكلمة داخل صورة الصفحة. تم تحسين التظليل من نقطة صغيرة إلى إطار أصفر/أزرق بحجم الكلمة مع padding بسيط، مع إضافة تمرير تلقائي لمعاينة الملف الأصلي عند اختيار كلمة.

الملاحظة الحالية: نافذة الاقتراحات تبقى مفتوحة أثناء الاختبار، لذلك يلزم تحديث/إعادة فتح الصفحة أو إغلاقها من الزر الظاهر قبل اختبار كلمة أخرى لا تقع تحت النافذة. كارد التحميل تم تعديله سابقاً ليظهر في تخطيط مركزي مستقل عند انتظار بيانات المراجعة.

بالنسبة لتسجيل الدخول عبر Google: الحزم الحالية لا تتضمن مكتبة Google OAuth، والتفعيل الحقيقي يحتاج Google OAuth Client ID وإعداد Authorized JavaScript origins/redirect URIs بحسب رابط الديمو أو الدومين النهائي. يمكن تجهيز الواجهة وزر Google، لكن المصادقة الفعلية لا تكون مكتملة بأمان بدون بيانات مشروع Google.
